CV Tailoring Settings

Configure settings for the CV tailoring service.

Tip: The CV tailoring service uses DeepSeek V3 API from OpenRouter to tailor your CV for each job. You can get a free API key from OpenRouter.

How to get an OpenRouter API key:

  1. Go to https://openrouter.ai/keys
  2. Sign up or log in with your account
  3. Create a new API key (the free tier includes access to DeepSeek V3)
  4. Copy the API key (starts with "sk-or-v1-...")
  5. Paste it in the field below

API Configuration

Your API key is stored locally and is only used to make requests to the OpenRouter API.

Note: The tailor service must be running at http://localhost:5000 for API key testing and CV tailoring to work. Make sure to start the service before testing your API key.

Tailor Prompt Customization

Tip: You can customize the prompt used to tailor your CV. The prompt is sent to the AI model along with your CV and the job description.

This prompt instructs the AI how to tailor your CV. Include placeholders: {original_yaml} for your CV and {job_description} for the job details.
